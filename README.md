# synesthesia

Personal archive of earth-science color and plotting scripts, spanning
2018–2026.

## Contents

- `colour.py`, `timeseries2color.py` — Python port of the Hughes &
  Williams (2010) CIE color loader and time-series-to-color mapping.
- `GMT.py` — GMT-based plotting helpers.
- `notation.sh`, `overflow.sh`, `projections.sh`,
  `setup_environment.sh` — shell utilities for the same workflow.
- `sealevel_spectra/` — sea-level spectra CSV data.

## Provenance

The original C++ implementation of Hughes & Williams (2010) lives in
the [GAIA](https://github.com/killett/GAIA) repository.

These scripts are preserved as-is for reference; they are not
maintained or packaged for reuse.
